快速开始

下载安装

git clone https://github.com/1920853199/go-blog.git

创建数据库，导入数据库文件

修改配置 conf/app.conf

#conf/app.conf
appname = go-blog
httpport = 8888
runmode = dev
EnableAdmin = false
sessionon = true
url = 127.0.0.1:8888
view = default

limit = 10
title = Go Blog
autograph = 如今的我，谈不上幸福，也谈不上不幸。

[db]

dbType = mysql
dbUser = root
dbPass = root
dbHost = 127.0.0.1
dbPort = 3306
dbName = blog

[redis]
rHost = 127.0.0.1
rPort = 6379

[wechat]
AppID = xxxxxxx
AppSecret = xxxxxxx
Token = xxxxxxx
EncodingAESKey = xxxxxxx

在项目根目录下执行 go run main.go

打开浏览器并访问 http://localhost:8888
